🍎 Apple MacBook Clone — 3D Interactive Website
A visually stunning, Apple-inspired product website built with React, featuring immersive 3D MacBook models rendered in real-time using Three.js and animated with GSAP. Styled entirely with Tailwind CSS for a sleek, minimal aesthetic that mirrors Apple's design language.

✨ Features

🖥️ 3D MacBook Model — Interactive 3D rendering powered by Three.js
🎬 Smooth Animations — Scroll-triggered and timeline animations via GSAP
⚛️ React Architecture — Component-based structure for scalability
🎨 Tailwind CSS Styling — Utility-first, responsive design system
📱 Responsive Layout — Optimized for all screen sizes


🛠️ Tech Stack

⚛️ React — UI framework & component architecture
🧊 Three.js — 3D model rendering & WebGL
🎬 GSAP — Scroll & timeline animations
🎨 Tailwind CSS — Styling & responsive design
